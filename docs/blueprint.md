# **App Name**: Exclusive Fashions

## Core Features:

- Product Catalog: Browse and filter fashion items (clothing, shoes, accessories).
- Product Details: View detailed information, media carousel, and availability for each product.
- Admin Dashboard: Manage products, categories, site settings, and user roles.
- Contact Form: Submit inquiries via a contact form and store in the database.
- Filtering and Sorting: Apply powerful filtering and sorting options on the products page.
- Authentication: Secure admin routes with Supabase Auth and role-based access control.

## Style Guidelines:

- Primary color: Vibrant Pink (#E91E63) to capture youthfulness and fashion-forward style.
- Background color: Light pink (#F8BBD0) provides a soft, clean backdrop.
- Accent color: Violet (#9C27B0) for highlighting important elements and calls to action.
- Headline font: 'Poppins' (sans-serif) for a contemporary and fashionable feel.
- Body font: 'PT Sans' (sans-serif) as it pairs well with Poppins and provides a modern yet readable feel.
- Use modern and minimalistic icons that align with fashion trends.
- Mobile-first, clean, and spacious layout with clear visual hierarchy.
- Subtle animations for transitions and interactive elements to enhance user experience.
